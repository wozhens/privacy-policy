# 网页采集助手隐私政策

更新日期：2026-06-29

网页采集助手重视用户隐私。本隐私政策说明扩展如何处理数据。

## 数据处理方式

网页采集助手用于从用户当前打开的网页中提取表格、列表、卡片和链接数据。所有提取、预览、字段编辑、数据清洗和文件导出均在用户浏览器本地完成。

## 我们不收集的数据

本扩展当前版本不会收集、上传、出售或分享以下数据：

- 用户浏览历史
- 网页正文内容
- 提取出的表格、列表、卡片或链接数据
- 导出的 CSV、Excel 或 JSON 文件
- 用户身份信息、账号信息或支付信息

## 本地保存的数据

为了提供模板复用能力，扩展会在浏览器本地存储中保存当前网站的提取模板和字段设置。这些数据仅保存在用户自己的浏览器中，不会上传到服务器。

用户可以在扩展中删除模板，也可以通过浏览器扩展管理或站点数据清理功能移除本地数据。

## 权限说明

扩展使用以下权限：

- `activeTab`：在用户主动点击扩展时访问当前标签页。
- `scripting`：向当前页面注入内容脚本，用于识别页面结构和提取数据。
- `storage`：在浏览器本地保存模板和字段设置。
- `<all_urls>`：允许用户在不同网站上使用网页采集功能。

## 第三方服务

当前版本不接入第三方分析、广告、AI 或远程数据处理服务。

## 联系方式

如需反馈问题或请求删除本地模板数据，请通过扩展商店页面提供的联系方式联系开发者。
